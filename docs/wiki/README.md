# Wiki Boundary

The wiki is part of the knowledge-base route. It is for broad academic
knowledge and links to structured literature notes, not for competition action
guidance.

Allowed content includes:

- PDE fundamentals;
- general Burgers equation background;
- neural operator concepts;
- operator learning concepts;
- FNO / DeepONet / PI-DeepONet concepts;
- summaries of academic ideas extracted from literature cards;
- links between concepts and citation metadata.

The wiki should not be used as the main place for SLURM, Kaggle, HDF5, Git, or
experiment-recording procedures. Those belong in skills, engineering workflows,
or tooling documents.

Do not add:

- competition-specific execution plans;
- concrete model-parameter adjustment paths;
- competition scoring-improvement advice;
- human-preloaded Agent action routes;
- long copied passages from papers.

Before adding a wiki page, check:

```text
docs/knowledge_base_route.md
docs/literature_library_policy.md
docs/preloaded_context_policy.md
```

Neutral official constraints belong in `docs/competition_clarifications.md`,
not in the wiki.
