# Wiki Boundary

本目录用于广泛 PDE、神经算子、科学计算和工程工具知识。知识库路线服务于通用背景整理，不承载赛题执行路线。

可以包含：

- PDE 基础；
- Burgers 方程一般背景；
- 神经算子一般原理；
- FNO / DeepONet / PI-DeepONet 一般概念；
- HDF5 / PyTorch / SLURM / Kaggle 一般知识；
- 通用科学计算、实验记录和工程工具知识。

不得包含：

- 针对本赛题的执行计划；
- 具体调参路线；
- 评分优化建议；
- 人工预置的 Agent 行动路线；
- 任何会把 broad knowledge 转化为赛题操作步骤的内容。

合作者新增知识页前应检查：

```text
docs/preloaded_context_policy.md
```

如需记录官方硬性规则、文件格式、时间限制、数据限制或提交要求，应写入 `docs/competition_clarifications.md`，不要写入 wiki。
