# Knowledge Base

This directory stores lightweight, structured Markdown knowledge artifacts for
the knowledge-base route.

Use it for literature cards, academic concept notes, reading notes,
taxonomies, and small metadata examples related to PDE, neural operators,
operator learning, Burgers equation background, FNO, DeepONet, PI-DeepONet, and
adjacent research topics.

Commit only lightweight Markdown, schemas, templates, scripts, tests, and small
example metadata. Do not commit PDFs, download caches, generated indexes,
vector stores, OCR dumps, large extracted text, or copied long passages.

Do not add competition-specific execution plans, concrete model-parameter
adjustment paths, competition scoring-improvement advice, or human-preloaded
Agent action routes.
