# Metadata Examples

This directory is for small example metadata records and schema examples for
literature cards and knowledge entries.

Examples should be small, synthetic or citation-style, and safe to commit.
They may demonstrate fields such as title, authors, arXiv identifier, DOI,
URL, topics, methods, summary, and citation text.

Do not commit PDFs, caches, large indexes, private files, or copied long paper
passages.

Do not add competition-specific execution plans or human-preloaded Agent action
routes.
