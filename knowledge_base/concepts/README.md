# Concepts

This directory is for reusable academic concept notes.

Allowed topics include general PDE concepts, neural operators, operator
learning, Burgers equation background, FNO, DeepONet, PI-DeepONet, and related
scientific concepts.

Do not use this directory for SLURM, Kaggle, Git, HDF5 engineering procedures,
or experiment-recording rules. Those belong in skills, engineering workflows,
or tooling documents.

Do not add competition-specific execution plans, concrete model-parameter
adjustment paths, or competition scoring-improvement advice.
