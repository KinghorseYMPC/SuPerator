# Literature Cards

This directory is for structured Markdown cards about individual papers.

Cards may include citation metadata, source links, paper summaries, research
questions, method summaries, findings, limitations, related concepts, and links
to local concept notes.

Do not commit PDFs, large extracted text, long copied excerpts, generated OCR
outputs, or download caches. Notes should summarize and cite.

Do not add competition-specific execution plans or human-preloaded Agent action
routes.
