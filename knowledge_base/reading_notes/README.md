# Reading Notes

This directory is for lightweight notes produced while reading papers or
surveying a general academic topic.

Reading notes may include short summaries, structured observations, source
tracking, citation metadata, open questions, and links to literature cards or
concept notes. Keep notes focused on reusable PDE, neural operator, operator
learning, Scientific ML, and AI4S knowledge.

Do not commit PDFs, large copied excerpts, OCR dumps, caches, generated
indexes, or vector stores.

Do not use reading notes for:

- Agent training logs;
- task logs;
- forged LLM traces;
- competition experiment records;
- current competition execution plans;
- model-choice, training-step, tuning-step, inference, or scoring guidance for
  Task 1 / Task 2.
